发表于 2012-11-23 | 分类于 MongoDB | | 阅读次数:

MongoDB 安装后默认不启用认证，也就是说在本地可以通过 mongo 命令不输入用户名密码，直接登陆到数据库，下面介绍下启用 mongodb 用户认证，详细如下：启用 mongodb 认证只需要在启动 mongod 服务时配置 auth 参数成 ‘true’即可可 ，在配置参数前先添加超级用户。
